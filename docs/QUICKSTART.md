# 快速开始指南 ⚡

## 🚀 5 分钟上手

### 步骤 1: 加载扩展

1. 打开 Chrome 或 Edge 浏览器
2. 访问扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 开启右上角的 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择 `chatgpt-sidebar-extension` 文件夹
6. 看到扩展出现在列表中 ✅

### 步骤 2: 打开 ChatGPT

访问 [https://chat.openai.com](https://chat.openai.com)，进入任意会话页面。

### 步骤 3: 查看侧边栏

右侧应该自动出现侧边栏，如果没有：

1. 按 `F12` 打开开发者工具
2. 查看 Console 中的日志
3. 确认是否在会话页面（URL 包含 `/c/`）

---

## 📚 核心功能快速体验

### 1️⃣ 目录导航 (2 分钟)

1. 在 ChatGPT 中连续提出 3-5 个问题
2. 侧边栏「📚 目录」Tab 自动显示所有提问
3. 点击任意条目 → 页面滚动到对应位置并高亮

### 2️⃣ 搜索功能 (1 分钟)

1. 在侧边栏顶部搜索框输入关键词（如"你好"）
2. 查看搜索结果列表
3. 点击结果跳转到对应消息

### 3️⃣ 创建项目 (2 分钟)

1. 切换到「📁 项目」Tab
2. 点击「+ 新建项目」
3. 输入项目名称（如"工作"）
4. 点击「添加到项目」，将当前会话加入项目

## 🔧 常见问题

### Q: 侧边栏没有出现？

**A:** 检查以下几点：
- 确认在会话页面（URL 包含 `/c/`）
- 刷新页面（`Ctrl/Cmd + R`）
- 打开开发者工具（`F12`）查看 Console 日志
- 确认扩展已启用（`chrome://extensions/`）

### Q: 目录没有更新？

**A:**
- 等待 0.5 秒（有 debounce 防抖）
- 刷新页面
- 查看 Console 是否有错误日志

### Q: 搜索不到内容？

**A:**
- 确认关键词拼写正确
- 搜索是大小写不敏感的
- 只搜索当前会话的消息

### Q: 如何查看存储的数据？

**A:** 在开发者工具 Console 中执行：
```javascript
chrome.storage.local.get(null, data => console.log(data));
```

### Q: 如何清空所有数据？

**A:**
- 方法 1: 侧边栏「⚙️ 设置」Tab → 点击「清空所有数据」
- 方法 2: Console 执行 `chrome.storage.local.clear()`

## 🎨 自定义

### 修改侧边栏宽度

编辑 `content/sidebar.css` 第 9 行：

```css
.sidebar-container {
  width: 400px;  /* 改为你想要的宽度 */
}
```

### 关闭开发日志

在各管理器文件中，将 `this.DEBUG = false`
